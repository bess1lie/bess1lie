# 🔒 bess1lie · detection‑only security tooling

Привет! Я разработчик из Алматы. Моя специализация — создание эффективных, быстрых и максимально точных инструментов для баг-хантеров и специалистов по кибербезопасности.

Я верю в подход **"Detection only, never exploitation"**: мои инструменты созданы для того, чтобы находить уязвимости, а не наносить вред. Все проекты разрабатываются с учетом границ области видимости (**scope-aware**) и всегда остаются открытым исходным кодом (**open source forever**).

---

## 🛠️ Мои проекты

| Проект | Описание | Ссылка |
| :--- | :--- | :--- |
| **🔍 apihunter** | Профессиональный CLI для тестирования безопасности REST API. Автоматизирует поиск OpenAPI, сканирование на IDOR/CORS и аудит аутентификации. | [GitHub](https://github.com/bess1lie/apihunter) |
| **🎯 bounthunt** | Инструмент для автоматизации разведки в программах Bug Bounty. Работает с поддоменами, эндпоинтами и фаззингом параметров. | [GitHub](https://github.com/bess1lie/bounthunt) |
| **🚀 gqlhunter** | Специализированный инструмент для поиска уязвимостей в GraphQL API через интроспекцию, фаззинг полей и анализ сложности запросов. | [GitHub](https://github.com/bess1lie/gqlhunter) |
